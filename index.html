API Documentation

Axel Springer is developing a general platform for distributing and providing content as a company-wide service. The key component of this platform is the iPool information hub that offers functionality for

  • storing
  • semantic enrichment
  • evaluation
  • retrieval
  • fully automated filtering

of various contents.

The main purpose of this page is to describe the structure and application of the iPool Frontend API embodying the user access tool for all content items stored and indexed within the iPool.

This is a work in progress page.

Documentation: https://www.dropbox.com/s/g1k63unkgh87oq9/iPool_API_V3_Documentation.pdf?dl=0
